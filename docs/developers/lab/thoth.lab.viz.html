thoth.lab.viz.init_notebook_mode(custom_css: list = None, custom_libs: dict = None, reload=False)[source]¶

Initialize notebook mode by linking required d3 libraries.

Parameters

  • custom_css –

    list of custom css urls to link

    NOTE: It is not possible to link locally located css files.

  • custom_libs –

    custom JavaScript libraries to link

    The libraries are linked using requirejs such as: `python require.config({ paths: {<key>: <path>} }); `

    Please note that <path> does __NOT__ contain .js suffix.

  • reload – bool, whether to re-initialize requireJS object
